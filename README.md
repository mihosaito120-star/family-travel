# family-travel

家族旅行ポータル用のGitHub Pagesサイトです。

## 置き方

おすすめは、新しいリポジトリ `family-travel` を作ることです。

1. GitHubで New repository
2. Repository name: `family-travel`
3. Public を選ぶ
4. Create repository
5. このZIPを展開して、中身を全部アップロード
6. Settings → Pages
7. Source: Deploy from a branch
8. Branch: main / root
9. Save

公開URLは通常、次の形になります。

`https://あなたのGitHub ID.github.io/family-travel/`

## トップページ専用リポジトリについて

GitHub IDと同じ名前のリポジトリ、たとえば `yourname.github.io` を作ると、

`https://yourname.github.io/`

がトップページになります。

最初は無理に作らなくて大丈夫です。まずは `family-travel` だけで始めるのがおすすめです。
